Optional: the documented system
If you want a complete, copy-paste executive operating system for running your day with ChatGPT (or another LLM), Spry Labs documents one approach here: Billionaire High Performance Coach (Gumroad).

This FAQ exists so the rest of the site can stay clean. If you’re deciding whether the documented system is for you, these are the practical questions that matter.

What exactly am I buying?

A one-time download: a written manual plus an executable prompt pack and quick-reference materials. It is designed to be copied into your own workflow and used inside ChatGPT (or another major LLM).

Is this a course? Do I get coaching calls?

No. It is a self-guided, static system. The “coaching” is an archetype and set of constraints you run through an LLM, not a human coach on calls.

Does this work with Claude, Gemini, Perplexity, and other models?

Yes. The system is written in plain language and uses prompts and operating rules that translate across major LLMs. Different models will respond differently, but the value is the structure and constraints you apply.

How much time does it take?

A typical one-time setup is about 20–30 minutes. Daily use is about five minutes—often less once you’ve internalized the loop.

Is this therapy or mental health advice?

No. It is educational and organizational. If you need clinical care, this is not a substitute.

I’m skeptical. What’s the smallest way to test it?

Use one answer page from this site as your “frame,” then ask your LLM to apply it to today with one constraint: choose fewer actions than you want to choose. If that immediately reduces cognitive load, you’re the target user.

Do I need to be consistent already?

No. The system is built around imperfect humans. It assumes missed days, fluctuating energy, and interruptions. The point is continuity, not perfection.

Is this only for business?

No. People use it for work execution, health routines, long-horizon identity goals, and “getting unstuck” in life admin. The system doesn’t care what your projects are; it cares that you can execute them without drama.
